# IMDB-Sentiment-Analysis-Using-RNN

## 📌 Overview
This project is a Deep Learning based NLP application that performs sentiment analysis on IMDB movie reviews using a Recurrent Neural Network (RNN) built with PyTorch.

The model predicts whether a movie review is:
- Positive 😊
- Negative 😞

This project helped me understand the complete NLP workflow including preprocessing, feature extraction, model building, training, and evaluation.

---

## 🚀 Features
- Data Cleaning and Preprocessing
- Lowercase Conversion
- URL Removal
- Punctuation Removal
- HTML Tag Removal
- Stopword Removal using NLTK
- Text Stemming
- TF-IDF Vectorization
- Custom RNN Architecture using PyTorch
- Model Training and Evaluation

## 📂 Dataset
Dataset Used:

Dataset Source:
https://ai.stanford.edu/~amaas/data/sentiment/

---

## 🔄 Project Workflow
1. Load Dataset
2. Clean and Preprocess Text Data
3. Remove Stopwords and Apply Stemming
4. Convert Text into Numerical Features using TF-IDF
5. Split Dataset into Train and Test Sets
6. Build Custom RNN Model
7. Train the Model
8. Evaluate Model Performance

---

## 📊 Model Performance
✅ Achieved approximately **82% accuracy** on test data.
